## 1. VIRTURA 空间茶话会 Vol.3

### 标题

`VIRTURA 空间茶话会 Vol.3｜AI 时代，我们怎么整理自己的作品、过程和知识`

### 建议时间

- 日期：`2026-03-29`
- 时间：`20:00 - 22:30`

### 建议形式

- `Space Salon / 数字沙龙 / 交流客厅`
- 小范围交流活动

### 一句话摘要

这次不聊空泛的 AI 口号，只聊更实际的数字归档和内容整理。

### 长说明

最近团队里不少人开始把作品、过程文件、旧项目、灵感碎片和仓库慢慢搬到 GitHub 上。真问题也跟着来了：文件太乱、版本太多、以前做过的东西找不到、别人也看不懂。

所以这次我们想聊的是更实际的事：数字归档、内容管理、仓库结构怎么搭，AI 在整理表格、补结构、写初稿这些地方能帮到什么，又有哪些判断只能靠人自己来做。

你可以带自己的仓库、文件夹或卡住的问题来。它不是一场“AI 万能论”的分享，更像一次认真聊方法、聊习惯、聊怎么把作品和知识整理下来的交流客厅。

### 可放的补充信息

- 地点：上海，具体地点报名通过后告知
- 人数：建议控制在 `10-15` 人

### 结尾 CTA

如果你想继续跟着后续茶话会、展映和测试走，最简单的方式是直接订阅：

`lu.ma/VIRTURA`

`Digital Library / Digital Stage 公开说明 02｜知识网络怎么整理成别人真的能用的入口`

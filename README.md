# VIRTURA-Collective

中文 | [English](./README-en.md)

`VIRTURA-Collective` is the team entry point of VIRTURA.

这里集中介绍 VIRTURA 作为团队正在做什么、已经公开了哪些作品与活动，以及外部可以从哪里继续了解或开始协作。

它不是资料仓总目录，也不是内部说明书，而是一份对外可直接阅读的团队公开入口。

这里默认成立的一件事是：团队协作和个人表达可以并行存在。加入 VIRTURA 不意味着把个人项目、个人研究和个人名字交给团队统一覆盖；更像是在一个朋友之间的协作网络里，围绕某些作品、活动和研究临时或长期形成合力。

## 目前已经公开的内容

- 3 条可稳定阅读的作品入口：`Drop Flow`、`TIMER`、`Observation and Symbiosis`
- 一条可以外部判断团队活跃度的活动时间线
- 第一批合作艺术家与协作者的公开名字档案
- 一组已经开始成形的研究方向与知识网络入口

## 团队目前主要在做什么

- 现场演出与舞台视觉
- 空间影像、展览与数字展演
- 数字舞台与在线发布
- 与创作直接相关的方法研究、知识整理与长期归档

## Tools / Systems / Network

VIRTURA 不只围绕作品本身展开，也在创作过程中逐步发展自己的工具、系统原型与知识网络，使实践能够从单一项目延展为可持续的结构。

这些工具与系统不是脱离作品的独立产品，而是在创作、演出、展览与研究的过程中逐步生成，并反过来支持作品的组织、预演、归档与协作。

当前最重要的三个方向包括：

- **Digital Stage / Scene System**  
  用于空间内容组织、状态预演与演出结构测试

- **Scene Viewer / Web Interface**  
  用于远程分享、场景查看、协作沟通与轻量化预览

- **Knowledge Network**  
  用于连接视觉、声音、空间、演出与方法研究之间的长期关系

同时，VIRTURA 也正在通过 ongoing stations 与 skill-based workflows，让提交、策展、归档、研究与展览运营逐步形成可复用的工作路径。

VIRTURA does not only develop works; it is also gradually building its own tools, system prototypes, and knowledge network through practice, allowing the work to evolve from isolated projects into a more sustainable structure.

These tools and systems are not separate products detached from artworks. They emerge through making, performing, exhibiting, and researching, and in turn support how works are organized, previewed, archived, and shared.

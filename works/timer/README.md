# TIMER Series

`TIMER` 是 VIRTURA 当前最具代表性的时间性音画互动系列之一。

它连接了实验原型、杭州首演、学术获奖、环形沉浸展览和未来空间化版本，是另一条非常清晰的系列线。

## Core Idea

`TIMER` 处理的核心不是单纯视觉效果，而是时间、节奏、空间中的感知控制。

这个系列的重要性在于：

- 以音画联动和时间结构为核心
- 已经在竞赛、展演、装置场景中验证
- 未来具备继续延展到沉浸式与空间版的可能性

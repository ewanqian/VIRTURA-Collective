# VIRTURA-Collective

中文 | [English](./README-en.md)

`VIRTURA-Collective` 是 VIRTURA 作为团队与创作网络的公开入口。

这里不只是团队介绍页，也不只是作品索引，而是一张正在持续展开的感知网络入口：从团队出发，向外连接合作艺术家、研究主线、活动、工作坊、现场实验与公开协作路径。

VIRTURA Collective 是一个以艺术家为主导的跨学科艺术团队与创意网络，主要关注空间图像、现场演出、视听结构以及感知经验的生成。他们通过演出、展览与公共项目，把视觉、声音和空间组织成可以被进入的现场，并持续在沉浸式空间视觉、多屏 / LED / 光影内容、现场视听与实时预演等方向展开合作与创作。

目前完整公开信息更新于如图 4、5、6 的 GitHub 平台 **VIRTURA-Collective**。它以团队为入口，向外延伸到合作艺术家、研究、活动、工作坊和公开协作路径，逐渐形成一张持续生长的创作网络。这里强调的也不是“统一品牌覆盖个人”，而是团队协作和个人表达并行存在的一种生态：分散的创作者、研究、演出与空间实验，被重新接入同一张感知网络之中。

在这张网络里，不同的人以不同方式进入、行动、停留，也会彼此留下痕迹，互相影响。它并不试图用强控制去统一所有方向，而更像是一种温和的连接：让作品、研究、现场、交流和协作在适当的时候重新发生关联，让行动本身逐渐构成网络的一部分。

除了研究、沙龙和内部交流工坊之外，目前也在继续探索 **On Site 实验场** 的模式，让更多的人因为场地、问题、项目与兴趣聚集在一起。他们会在工厂里写诗，在图书馆举办音乐会，在海滩烹饪，在超市里 VJ，在高速公路上举办公共集会，让现场成为创作的一部分，也让空间重新成为人与人之间发生联系的媒介。

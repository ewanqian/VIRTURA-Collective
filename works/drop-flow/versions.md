# Drop Flow Versions

这页把 `Drop Flow` 的版本关系压缩成可快速理解的索引。

## Evolution Path

`Drop Flow` 目前可以概括为这条演进路线：

1. 技术原型
2. 内部实验
3. 公开展演
4. 竞赛与大型展览适配
5. 面向未来的空间化规划

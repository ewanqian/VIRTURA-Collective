# Artists

这里整理 VIRTURA 已公开可识别的合作艺术家、创作者与协作者档案。

这份目录首先用于公开识别与致谢，也方便外部从“项目”回到“人”，再从“人”回看他参与过的系列、版本与合作方向。

它不是固定编制名单，而是一组可以继续生长的名字档案。这里既包括在团队作品里反复出现的长期合作者，也包括保留个人项目和个人作者身份、但与团队形成稳定互相支持的人。

## 这份目录怎么用

- 想先按名字认识团队协作网络：直接看下面的第一批档案
- 想从项目回头看人：在各系列的 `collaborators.md` 里再跳回这里
- 想判断某位协作者适合哪类合作：先看“主要方向”

## 后续会继续补什么

- 每位协作者参与过的具体版本和时间线
- 公开可引用的外部资料、活动页面或展览记录
- 与项目相关的个人思考、短自述或公开发言
- 团队视角下可稳定引用的个人简介

如果本人愿意公开补充，也欢迎继续增加：

- 自己更认可的公开简介
- 与项目相关的思考或方法笔记
- 可公开引用的外部链接

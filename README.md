# VIRTURA-Collective

`VIRTURA-Collective` is the team entry point of VIRTURA.

它承载团队介绍、成员、代表作品、研究方向与合作入口，是 VIRTURA 对外最清晰的团队主仓。

## What It Is

这里不是工具总仓，也不是世界观母仓。

这里负责回答 5 个最基本的问题：

1. VIRTURA 是谁
2. 谁在参与
3. 团队做过什么
4. 团队在研究什么
5. 如何合作与进入

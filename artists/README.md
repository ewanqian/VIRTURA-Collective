# Artists

这里整理 VIRTURA 已公开可识别的合作艺术家、创作者与协作者档案。

这页不是固定编制名单，而是一组可以继续生长的个人入口：

- 先认识这个人主要负责什么
- 再看他参与过哪些系列或版本
- 后续继续补项目相关思考、公开资料和合作脉络

## 这页怎么用

- 想先按名字认识团队协作网络：直接看下面的第一批档案
- 想从项目回头看人：在各系列的 `collaborators.md` 里再跳回这里
- 想判断某位协作者适合哪类合作：先看“主要方向”

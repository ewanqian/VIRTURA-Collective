# TIMER Assets

这页说明 `TIMER` 系列后续如何整理海报、证书、截图和视频封面。

## Suggested Structure

```text
timer/
  README.md
  versions.md
  public-materials.md
  awards-and-presentations.md
  collaborators.md
  assets.md
  manifest.json
  assets/
    _inbox/
    stage/
    poster/
    social/
    docs/
```

## Recommended Asset Types

- `stage/`: 现场图、展映图
- `poster/`: 海报、KV
- `social/`: 视频封面、社交媒体截图
- `docs/`: 奖状、证书、官方页截图

## Current Policy

当前在 `VIRTURA-Collective` 中优先放：

- 索引
- 小样
- 公开可引用资料
- 与奖项和展演关系直接相关的文件

大体积工程和原始母文件以后再决定去向。

# Observation And Symbiosis

`Observation and Symbiosis` 是 VIRTURA 当前已公开的展览型项目线之一。

它与 `Drop Flow`、`TIMER` 的不同在于，它更明确地站在“展览 / 数据景观 / 数字水墨”这条线上，而不是现场演出主导。

## Core Idea

这个项目基于地理数据与动态粒子系统，把严谨的数据结构转化为持续流淌的数字景观。

它的重要性在于：

- 它连接了数据可视化与沉浸景观
- 它体现了 VIRTURA 对“数字自然”的另一条表达路径
- 它已经在多个展览场景中落地

# About

`VIRTURA-Collective` 是 VIRTURA 的团队主入口。

它面向的是团队访客、潜在合作方、策展人、媒体、同行与未来成员。

## Core Definition

VIRTURA 是一个去中心化的艺术协作网络。

它不是行政化的固定编制团队，而是围绕作品、研究、演出、空间实验与数字舞台持续协作的创作共同体。

## Why This Repository Exists

过去团队介绍、作品入口、组织说明、研究方向和导航关系分散在不同仓库中，外部访客很难快速理解：

- 团队本体是什么
- `SpacePort` 是什么
- 哪些内容是作品
- 哪些内容是研究或工具

这个仓库的存在，就是为了把“团队本体”稳定下来。

## What Belongs Here

- 团队介绍
- 成员与参与方式
- 代表作品与演出入口
- 核心研究方向
- 合作说明

## What Does Not Belong Here

- 工具主仓角色
- 世界观总控壳
- 所有文章发布
- 内部治理细节

这些内容会分别留在相关仓库中，通过互链建立关系。

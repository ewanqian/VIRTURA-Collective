# Planned

这里整理 `VIRTURA` 当前已经在排期中的活动与发布节点。

这一页不是最终报名页，而是一个“接下来会发生什么”的总览页。

真正对外开放的报名、申请和开幕入口，以 [VIRTURA 官方活动日历](https://lu.ma/VIRTURA) 为准。

## 当前最接近落地的 4 个节点

### 1. VIRTURA 空间茶话会 Vol.3｜AI 时代，我们怎么整理自己的作品、过程和知识

- 日期：`2026-03-29`
- 形式：交流客厅 / 小范围社群互动
- 当前口径：
  不聊空泛的 AI 口号，只聊更实际的事：作品、过程文件、旧项目、灵感碎片和仓库，到底该怎么整理，AI 在这里究竟能帮上什么，哪些判断还是得自己来做。
- 报名方式：
  [Luma 活动页](https://luma.com/jy5fef4u)

### 2. MOVA 展映 01｜我们第一次把 GitHub 里的内容整理成数字展览

- 日期：`2026-04-03`
- 形式：线上展映开启
- 当前口径：
  用一次很轻量的展映，测试 GitHub 里的作品、研究、活动记录和页面，能不能被组织成一条自然的观看路线。
- 公开入口：
  [Luma 活动页](https://luma.com/kw41rtby)
- 配套入口：
  [MOVA 试运行展](https://github.com/ewanqian/VIRTURA-SpacePort/tree/main/stations/mova/exhibitions/library-pilot-2026-03)

### 3. 在制内容审计会 01｜把正在做的东西拿出来，我们一起挑问题

- 日期：`2026-04-04`
- 形式：小范围审计 / 批评 / 互相挑问题
- 当前口径：
  如果你最近在做页面、视觉、声音、空间内容，或者只是手上有一个怎么都整理不顺的东西，欢迎带来。我们就做一件事：一起看问题到底出在哪。
- 公开入口：
  [Luma 活动页](https://luma.com/3be89q8l)

### 4. Digital Library / Digital Stage 公开说明 01｜AI 时代怎么把作品、材料和过程整理成能继续工作的库

- 日期：`2026-04-10`
- 形式：线上公开说明
- 当前口径：
  从 GitHub 上的仓库、归档、知识网络和页面前台出发，讲清一套内容结构怎么真正长成能继续工作的库，而不是只看起来整齐。
- 公开入口：
  [Luma 活动页](https://luma.com/kjyluf41)

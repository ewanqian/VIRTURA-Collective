# Activities

这里整理 VIRTURA 的公共活动、沙龙、工作坊与线上线下时间线。

如果你是从小红书、朋友转发或一次活动认识 VIRTURA 的，这页应该帮你快速确认：团队不是只在写概念，而是真的在持续发生事情。

它和 [Performances](../performances/README.md) 的关系很近，但重点不同:

- `Activities` 更关注团队如何与公众、同行和未来协作者发生连接
- `Performances` 更关注作品在具体场域中的呈现

## Space Salon 这条活动线

当前最适合外部先认识的一条活动线，就是 `Space Salon / 数字沙龙`。

- 它承接的是更轻量、可持续的交流与分享
- 它既可以连接作品和研究，也可以连接后续的合作与公开活动
- 目前已经有 `Vol.0`、`Vol.1` 两个已公开节点，`Vol.2` 正在筹备中

## How People Usually Enter

很多协作关系并不是从正式“加入申请”开始，而是从这些活动场景进入:

- 先参加一次公开活动或线上分享
- 再围绕某个系列、某个研究方向或某个小模块发生协作
- 最后根据节奏，转为项目制或长期协作

这也是为什么活动页很重要：它是很多人第一次真正靠近团队的入口。

## If You Want To Join

目前更适合的进入方式有四种:

1. 围绕某个项目系列参与一次清晰的协作
2. 围绕研究、写作或知识网络贡献一个小模块
3. 围绕工具、模板或发布结构参与搭建
4. 先从活动、沙龙和公开交流建立连接

更具体的参与说明见: [Collaboration](../collaboration/README.md)

如果你还不确定自己适合哪种方式，最简单的做法是先从一场活动或一条研究线开始。

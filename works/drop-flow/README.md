# Drop Flow Series

`Drop Flow` 是 VIRTURA 当前最具代表性的系列作品之一。

它跨越了原型实验、内部展演、公开现场、竞赛获奖、双年展开幕式与后续空间化规划，是团队最适合优先沉淀为系列档案的项目线。

## Core Idea

`Drop Flow` 的核心不是一次性爆发，而是从一次坠落、一种流动和一种感知状态出发，逐层扩展为持续变化的空间叙事。

这也是这个系列最重要的特征：

- 不同版本之间有连续关系
- 同一创作能进入不同媒介与场景
- 作品本身可以继续生长为 Digital Stage 与资产化入口

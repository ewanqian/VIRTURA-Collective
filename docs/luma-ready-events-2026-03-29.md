## 1. VIRTURA 空间茶话会 Vol.3

### 标题

`VIRTURA 空间茶话会 Vol.3｜带一段最近在做的东西，来聊聊空间、作品与正在发生的事`

### 建议时间

- 日期：`2026-03-29`
- 时间：`20:00 - 自由散场`

### 建议形式

- `Space Salon / 数字沙龙 / 交流客厅`
- 小范围交流活动

### 一句话摘要

这不是一场说明会，也不是一场正式发布会，而是一次更偏交流的空间茶话会。

### 长说明

如果你最近正在做视觉、声音、空间、网页、互动或其他还在生长中的数字内容，欢迎带一段最近在做的东西来。

它可以是一段视觉、一段音乐、一个场景草图、一个页面、一个半成品，或者只是一个你最近一直想继续聊的问题。

这次我们想把更多时间留给真实交流、互相观看和顺着现场继续展开的话题，而不是做成单向介绍。你不需要准备完整作品，也不需要先有明确合作目标，只要你愿意把最近正在发生的事带进来就可以。

### 可放的补充信息

- 地点：上海，具体地点报名通过后告知
- 人数：建议控制在 `10-15` 人
- 参与方式：以 [VIRTURA 官方活动日历](https://lu.ma/VIRTURA) 发布入口为准

### 结尾 CTA

如果你想继续跟着后续茶话会、测试和线上展映走，最简单的方式是直接订阅：

`lu.ma/VIRTURA`

## 2. MOVA 开幕｜从库到展：VIRTURA 数字空间研究摘编

### 标题

`MOVA 开幕｜从库到展：VIRTURA 数字空间研究摘编`

### 建议时间

- 日期：`2026-04-03`
- 时间：`20:30`

### 建议形式

- `MOVA / Museum of Virtual Art`
- 线上展映开启

### 一句话摘要

一次很轻量的线上展映试运行，测试 GitHub 里的内容能否被组织成一次自然的观看过程。

### 长说明

这不是一次大型线上发布会，而是一场试运行展。

我们会把已经公开的作品、研究、知识网络和活动归档，先整理成一条可以边走边看的观看路线。它更像一次线上展览的第一版样板，也是在测试 `MOVA` 作为公共发布线是否成立。

如果你对数字空间、线上展映、作品档案和研究如何进入公共前台感兴趣，这会是一个很合适的入口。

### 配套入口

- [MOVA 试运行展](https://github.com/ewanqian/VIRTURA-SpacePort/tree/main/stations/mova/exhibitions/library-pilot-2026-03)

### 结尾 CTA

看完这次试运行展后，如果你想继续跟后续节点，欢迎直接订阅：

`lu.ma/VIRTURA`

## 3. Practice Night 01

### 标题

`Practice Night 01｜带一段半成品来，我们一起看怎么继续`

### 建议时间

- 日期：`2026-04-04`
- 时间：`20:00`

### 建议形式

- `Practice / 开放实践`
- 半成品共看 / 小范围交流

### 一句话摘要

带一段视觉、一段声音、一个页面草稿，或者一个还没想通的问题来。

### 长说明

这是一场准备压力很低的开放实践夜。

你不需要带完整作品来，只要带一个还在路上的东西就可以。我们会一起看它现在处在什么状态、还能往哪里走，以及哪些问题值得继续追。

它适合成长中的创作者，也适合刚刚开始靠近这个网络的人。

### 结尾 CTA

如果你更想长期跟着 `Practice / Space Salon / 太空气球 / MOVA` 这些线走，建议直接订阅：

`lu.ma/VIRTURA`

这场更适合对展览、现场、记录、存档和线上转译感兴趣的朋友来听，也适合那些正在思考“活动结束之后还剩下什么”的人。

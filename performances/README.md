# Performances

这里收录 VIRTURA 已公开可识别的现场、舞台、展演与展览型作品入口。

相比一般作品页，这里更关心作品在什么场域发生、以什么形式出现，以及哪些版本已经进入真实现场。

## 这页与 `Works`、`Activities` 的区别

- [Works](../works/README.md) 更偏向精选作品与系列入口
- `Performances` 更偏向作品如何在现场、展览和事件里发生
- [Activities](../activities/README.md) 更偏向团队公共活动、沙龙、工作坊与交流时间线
